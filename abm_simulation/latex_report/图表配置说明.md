## 🎨 图表设置细节

### 统一格式
- **宽度**: `0.95\textwidth`（几乎占满全文宽度）
- **位置**: `[H]`（强制固定在代码位置）
- **对齐**: `\centering`（居中对齐）
- **题注**: 包含子图说明 `(a)... (b)... (c)...`

## 📍 位置安排原则

### 最佳实践
1. **结果分析之后**：在文字描述完主要发现后插入
2. **本节小结之前**：在小结前给读者直观印象
3. **逻辑连贯处**：不打断叙述流程

## 🔧 技术实现

### 使用的宏包
```latex
\usepackage{float}      % 提供 [H] 选项
\usepackage{graphicx}   % 提供 \includegraphics
\usepackage{subcaption} % 支持子图（可选）
```

### 图表环境
```latex
\begin{figure}[H]  % [H] 强制固定在当前位置
    \centering     % 居中
    \includegraphics[width=0.95\textwidth]{图片路径}
    \caption{题注文字}
    \label{标签}
\end{figure}
```

### 引用方式
```latex
如图~\ref{fig:baseline_summary}所示...
```

## ⚠️ 注意事项

### 编译时
1. 确保所有图片文件存在
2. 检查图片路径是否正确
3. 使用 XeLaTeX 编译（支持中文和 PNG）

### 修改时
1. 不要删除 `[H]` 选项（会导致图表乱跑）
2. 保持宽度一致（0.95\textwidth）
3. 题注要详细但不冗长

### 打印时
1. 建议使用彩色打印
2. 确保图片清晰度
3. 检查灰度转换效果
