## 🔧 编译方法

### 方法 1：使用编译脚本（推荐）

Windows 用户直接双击运行：
```

```

### 方法 2：手动编译

在终端执行以下命令：

```bash
cd latex_report

# 第一次编译
xelatex main.tex

# 第二次编译（解决引用和目录）
xelatex main.tex
```

## 📋 系统要求

### 必需的 LaTeX 环境

需要安装以下任一 LaTeX 发行版：

- **MiKTeX**：https://miktex.org/
- **CTeX**：http://www.ctex.org/

### 必需的宏包

文档使用的主要宏包：
- `ctex` - 中文支持
- `geometry` - 页面设置
- `graphicx` - 图形插入
- `booktabs` - 专业表格
- `hyperref` - 超链接
- `amsmath` - 数学公式
- `float` - 浮动体控制
- `subcaption` - 子图支持

以上宏包在所有主流 LaTeX 发行版中都已包含。

## ⚠️ 常见问题

### Q1: 编译时提示找不到 ctex 宏包
**解决方案**：安装完整的 TeX Live 或 CTeX 套装

### Q2: 图表无法显示
**解决方案**：
1. 检查 `../results/all_figures/` 目录是否存在
2. 确认图表文件名称匹配
3. 尝试使用相对路径或绝对路径

### Q3: 中文显示为乱码
**解决方案**：
1. 确保使用 XeLaTeX 编译（不是 pdfLaTeX）
2. 检查 .tex 文件编码为 UTF-8
3. 安装中文字体（SimHei, Microsoft YaHei 等）

### Q4: 引用显示为 ??
**解决方案**：编译两次即可解决
