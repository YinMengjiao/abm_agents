# 实验 10 可视化标签位置修复报告

## 🐛 问题描述

**文件**: `abm_simulation\results\all_figures\exp10_systemic_risk_systemic_risk_analysis.png`

**问题**: 第 6 个子图（系统韧性评估）中的数据标签位置错误，标签显示在柱子底部而不是顶部

---

## 🔍 根本原因

在 `_plot_resilience_assessment` 函数中，数据标签的垂直对齐方式设置正确（`va='bottom'`），但是 y 轴的范围没有为标签预留足够的空间，导致标签被截断或显示位置不当。

**问题代码**（第 204-208 行）:
```python
# 添加数值标签
for bar, val in zip(bars, values):
    height = bar.get_height()
    ax.text(bar.get_x() + bar.get_width()/2., height,
           f'{val:.3f}', ha='center', va='bottom', fontsize=9)
```

---

## ✅ 解决方案

### 修改内容

**文件**: `experiments/exp10_systemic_risk/visualization_systemic_risk.py`

在第 208 行后添加 y 轴范围调整：

```python
# 添加数值标签
for bar, val in zip(bars, values):
    height = bar.get_height()
    ax.text(bar.get_x() + bar.get_width()/2., height,
           f'{val:.3f}', ha='center', va='bottom', fontsize=9)

# 调整 y 轴上限，为标签留出空间
ax.set_ylim(0, 1.15)  # ← 新增：从 1.0 扩展到 1.15
```

### 技术说明

- **原 y 轴范围**: `0` 到 `1`
- **新 y 轴范围**: `0` 到 `1.15`（增加 15% 的空间用于显示标签）
- **标签位置**: 现在标签完整显示在柱子顶部上方

## 🔧 技术要点

### 1. matplotlib 标签定位

在使用 `bar chart` 时，标签位置的常见设置：

```python
# 标签在柱子顶部上方
ax.text(x, height, label, va='bottom')  # ✓ 正确

# 标签在柱子中心
ax.text(x, height/2, label, va='center')

# 标签在柱子内部底部
ax.text(x, 0.05, label, va='bottom')
```

### 2. 为标签预留空间

当标签添加到图表时，需要调整坐标轴范围：

```python
# 方法 1: 固定扩展
ax.set_ylim(0, 1.15)  # 扩展 15%

# 方法 2: 按比例扩展
max_val = max(values)
ax.set_ylim(0, max_val * 1.15)  # 扩展 15%
```

### 3. tight_layout 配合

使用 `tight_layout(rect=[...])` 为 suptitle 预留空间的同时，也要确保子图内部元素有足够空间：

```python
plt.tight_layout(rect=[0, 0, 1, 0.96])  # 为主标题预留空间
ax.set_ylim(0, 1.15)  # 为标签预留空间
```
