## 论文表述模板

```
基准实验中,耦合强度 J 从 0.20 线性增长至 0.35。通过独立 J 扫描实验(N=500, 
网格 ΔJ=0.01,每点 20 次独立运行)测定,修改版 Ising 模型在 N=500 下的临界耦
合强度约为 J_c ≈ 0.46,显著高于平均场近似值 1/⟨k⟩ ≈ 0.17。这一偏移可归
因于我们对标准 Glauber 动力学的三处修改(L3 稳定性、L2/L4 回归概率、极
端态单步约束),它们共同抑制了自旋对齐倾向。

因此基准实验全程处于无序相(J < J_c),20 次独立运行的平均磁化强度为 
⟨M⟩ = X.XX ± Y.YY,与有限尺寸无序相理论预期(σ_M ≈ 1/√N ≈ 0.045)一致。
依赖等级分布以 L3(半委托型)为主导。

相变性质方面,我们观察到磁化强度在 J ≈ 0.46 附近显著上升,磁化率 χ 持
续增长至 J = 0.50 附近,Binder 累积量 U 从 0.18 单调上升至 0.66。这些
特征表明系统经历了一个连续转变过程,但有限尺寸效应使得精确判断相变阶数
需要更大的系统规模(详见 Supplementary)。

这一设计有意避免系统跨越相变点,为 Exp10(系统风险实验)中将 J 推过 J_c 
的对照场景提供清晰的无序相基线。
```
