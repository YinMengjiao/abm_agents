# 数据驱动仿真初始化指南

## 📊 概述

本系统现已支持**从真实调查数据初始化仿真**，确保仿真初始状态与现实世界一致。

### 数据流

```
ACDDS问卷调查 (N=2000)
    ↓
熵权法计算综合得分
    ↓
L1-L5等级划分
    ↓
仿真初始化 (initial_level_distribution)
    ↓
Ising-D-I-B动态演化
    ↓
政策实验与干预分析
```

## 🚀 使用方法

### 方式1：使用调查数据（默认推荐）

```bash
cd abm_simulation
python run_simulation.py
```

或显式指定：

```bash
python run_simulation.py --mode survey
```

**效果**：
- 自动读取 `data_survey/acdds_results.csv`
- 统计L1-L5真实分布
- 用该分布初始化500个消费者Agent
- 运行300步仿真

### 方式2：交互式手动输入

```bash
python run_simulation.py --mode interactive
```

**效果**：
- 程序会提示你输入L1-L5的比例
- 支持自动归一化
- 适合敏感性分析

### 方式3：理论默认分布

```bash
python run_simulation.py --mode default
```

**效果**：
- 使用对称分布：L1=10%, L2=25%, L3=30%, L4=25%, L5=10%
- 适合对比实验

## 📝 代码调用示例

在其他实验中使用调查数据：

```python
from simulation import SimulationConfig, ABMSimulation

# 加载调查数据
survey_dist = SimulationConfig.load_survey_distribution()

# 创建配置
config = SimulationConfig(
    n_consumers=500,
    initial_level_distribution=survey_dist,
    # ... 其他参数
)

# 运行仿真
sim = ABMSimulation(config)
sim.run()
```

## ⚠️ 注意事项

1. **数据文件路径**：确保 `data_survey/acdds_results.csv` 存在
2. **等级名称映射**：CSV中的等级列必须包含 'L1', 'L2', 'L3', 'L4', 'L5'
3. **回退机制**：如果调查数据文件不存在，系统会自动使用默认理论分布
